    Utilities.grab(obj, key) → Mixed

    Removes given key from object and returns value of removed key, similar to Ruby's Object#delete.

    Example
    var obj = { k1: 'abc', k2: 'def' };
var k1  = Utilities.grab(obj, 'k1');
 inspect(k1);
// -> 'abc'
 inspect(obj);
// -> { k2: 'def' }
