    Utilities.patchBuffers(original, patch1[, patch2[, patchN]]) → Buffer

    Applies patches to original buffer. Each patch should be a Buffer with unified patch, e.g.:

    @@ -1,3 +1,3 @@
 123
-455
+456
 789

    Patches are applied from left to right (first patch1, then patch2, and so on). Each patch (except first one) is applied on the result of patching with previous one.
